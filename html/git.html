Git

  • git init – инициализирует новый локальный репозиторий в текущей папке.
  • git clone [url] – копирует существующий удаленный репозиторий на ваш компьютер.
  • git status – показывает текущее состояние рабочего каталога и отслеживаемые файлы.
  • git add [файл] – добавляет измененные или новые файлы в индекс для следующего коммита.
  • git commit -m "[сообщение]" – фиксирует сохраненные изменения и создает снимок истории проекта.
  • git push – отправляет ваши локальные коммиты в удаленный репозиторий на сервере.
  • git pull – скачивает и объединяет изменения из удаленного репозитория в вашу локальную ветку.
